# EHCP - A parents guide.
           
           ## Description

           A full stack application with a complete back-end solution with CRUD operations connected to a MySQL database.
            
           ## Installation 

           Git clone to your local machine, npm install. Copy .env.example file to .env file and add your own password.
   
           ## Useage 
           
           npm run start in console to run the application. Application can be opened in local host, user is able to register and log in as a user, to create, edit and delete their own blog posts as well as view previously created posts.
